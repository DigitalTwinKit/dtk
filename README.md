# DigitalTwinKit — Free Claude Skills

Three free Claude skills for your digital twin video pipeline. Upload these to a Claude Project and they work automatically.

## Setup

1. Create a new **Project** in Claude (claude.ai → Projects → New project)
2. Upload all `.md` files from the `skills/` folder as **project knowledge**
3. Start a conversation in the project — Claude reads the skills automatically

## Skills

| File | What it does |
|---|---|
| `script-writer.md` | Write a video script in your voice from a topic + format + audience |
| `hooks.md` | Generate 10 alternative opening hooks for a draft script |
| `disclose.md` | Draft AI-disclosure copy for video descriptions, proposals, or bios |

# DigitalTwinKit — Free Claude Skills

Three Claude Code skills for your digital twin video pipeline. Write scripts in your voice, generate opening hooks, and draft AI disclosure language — all from slash commands.

## Install

```
/plugin marketplace add DigitalTwinKit/dtk
/plugin install dtk
```

## Skills

| Command | What it does |
|---|---|
| `/dtk:script` | Write a video script in your voice from a topic + format + audience |
| `/dtk:hooks` | Generate 10 alternative opening hooks for a draft script |
| `/dtk:disclose` | Draft AI-disclosure copy for video descriptions, proposals, or bios |

## Requirements

- [Claude Pro](https://claude.ai) with Claude Code enabled
- A persona document (optional but recommended — the skills match your voice when one is loaded)
